# Assets

## Instrucciones

### Imagen de fondo del Hero

Agrega tu imagen de fondo en esta carpeta con el nombre: **`hero-bg.jpg`**

La imagen aparecerá detrás del buscador en la página principal con un overlay rojo semitransparente.

**Ubicación esperada:** `assets/images/hero-bg.jpg`

**Recomendaciones:**
- Formato: JPG o WebP
- Tamaño recomendado: 1920x800px mínimo
- Peso optimizado: < 500KB
- Buena resolución para que se vea nítida en pantallas grandes

Si cambias el nombre del archivo, actualiza la ruta en `index.css` línea ~216:
```css
url('assets/images/hero-bg.jpg')
```
